Getting started
===============

Prepare etcd
------------

Sabakan requires [etcd][].  Install and run it at somewhere.

Install sabakan and sabactl
---------------------------

Install `sabakan` and `sabactl`:

```console
$ go get -u github.com/cybozu-go/sabakan/cmd/sabakan
$ go get -u github.com/cybozu-go/sabakan/cmd/sabactl
```
